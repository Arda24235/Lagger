
## **Lagger**
### Lagger is a computer prank for your friends.
---
In Lagger, selecting the "hide" checkbox hides the window.
Using the Lag V1 button uses a lot of CPU. Selecting V2 opens and closes many windows and uses a lot of RAM.
On the "Annoying" side, the "Spam" feature opens the program's "About" dialog box too often,
and "Start Menu Spam" opens the start menu every 3 seconds.
